Sur moi

Bienvenue!

Salut! comment ça va? Je m'appelle Sergio.

Je suis un autodidacte passionné et étudiant de la carrière de Informatique à l'Université Nationale de Cordoba.

J'ai réalisé quelques projets Web, en utilisant dans le FrontEnd : HTML5, CSS3, Grid, Flexbox, Bootstrap, TypeScript, JS, Angular et Bootstrap, tout responsive design, dans la partie base de données SQL Server et MySQL la connectant au BackEnd via MVC, en consommant tout ApiRest que vous avez créé, écrit en C # via ASP.Net, ou Python via Django, et le communiquer avec des requêtes HTTP en JSON, j'utilise également CORS pour connecter le Frontend et le Backend.

En tant que QA Tester, j'ai conçu et réalisé des tests exploratoires, de régression, de fumée et automatisés avec Selenium, Behave et Python, pour des scénarios reproductibles, augmentant la couverture de test dans l'application (Web, Mobile ou Desktop).

Mes compétences

Certaines des technologies que j'ai utilisées et que j'apprends activement

QA Testing

Conception et exécution de cas de tests, tests exploratoires, régressions, tests de fumée, rapports de bogues, tests manuels et automatiques avec Selenium, Behave, Gherkin et Python.

Angular

Angular est un framework Javascript axé sur le front d'une application, il a une structure SPA donc il est très rapide, mais en même temps très complet et a une grande communauté.

Bootstrap

C'est un framework CSS qui contient des modèles de conception, formulaires, boutons, boîtes et autres éléments de conception basée sur HTML et CSS pour le développement FrontEnd.

SQL Server

Ici, je crée, gère et administre les bases de données, ainsi que leurs procédures stockées pour faciliter les requêtes envoyées du Fronted au BackEnd.

JIRA

C'est un puissant logiciel de gestion de projet, je peux voir les tableaux Kanban de n'importe où, activer les notifications, visualisez les métriques des équipements et avec Zephyr automatisez les cycles de test.

GIT-- GITHUB-- GITLAB

Git est un logiciel de contrôle de version, de travailler sur un code stable entre plusieurs développeurs, en ajoutant du code de n'importe où dans le monde.

ReactJs

React est une bibliothèque JavaScript, je m'exerce avec cet outil, qui est parmi les plus demandés Du marché.

Android Studio

Tests dans des applications Android natives, erreurs de débogage et création de versions.

JavaScript

C'est le langage orienté objet le plus utilisé, il permet effectuer des effets et des actions dynamiques, je vais bientôt pratiquer Node Js.

HTML5

HTML est un langage de balisage hypertexte qui utilise des balises ordonné forme la structure d'une page Web.

CSS3

CSS (feuilles de style en cascade) styliser le code HTML, référencer, donner des effets et schématiser ses éléments (Grid ou Flexbox).

Responsive Design

Responsive Design permettant à nos sites Web de pouvoir s'adapter à n'importe quel appareil mobile ou de bureau.

Python

C'est un langage très populaire, utile et puissant, je l'utilise généralement pour écrire du code pour des tests d'automatisation, avec Selenium, Behave and Cucumber (écrit en Gherkin).

Wordpress

C'est un système de gestion de contenu, développé en PHP, c'est l'un des principaux outils pour la création de pages web commerciales.
